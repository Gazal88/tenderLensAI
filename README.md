# Tender Evaluator & Assisant

A modern Next.js application that automates the evaluation of vendor submissions against tender documents using AI (Groq API).

## Features

- **Automated Tender Evaluation**: Upload a main tender document and multiple vendor submission documents to automatically generate compliance scores, pass/fail eligibility, risk assessments, and human-like explanations.
- **TenderBot**: An integrated, interactive chat widget powered by the Groq API (Llama 3) to answer queries, explain platform usage, and assist users in real-time.
- **Beautiful UI**: Built with Tailwind CSS and Next.js, featuring a clean, responsive, and animated user interface.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js 14/15](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **AI Integration**: [Groq API](https://groq.com/) (using Llama 3 models)
- **Icons**: [Lucide React](https://lucide.dev/)
